# CryptoEmperor AI

**加密货币交易信号系统** - 基于技术指标的自动化交易信号生成、风控管理和日报��出。

## 🚀 功能特性

- ✅ **实时信号生成**：基于 RSI 指标的买入/卖出/观望信号
- ✅ **多币种支持**：BTC、ETH、BNB、SOL 等主流币种
- ✅ **风控模块**：仓位管理、止损、日亏损闸门
- ✅ **信号记录**：JSON 持久化，历史信号可追溯
- ✅ **日报生成**：Markdown 格式，包含统计分析
- ✅ **回测引擎**：验证策略历史表现
- ✅ **配置化**：YAML 配置文件，易于调整参数
- ✅ **日志系统**：按日期分文件，自动清理旧日志
- ✅ **Telegram 推送**：实时信号通知、日报推送
- ✅ **定时调度**：自动化运行，无需人工干预
- ✅ **Docker 部署**：一键容器化部署

## 📦 安装

### 1. 克隆仓库
```bash
git clone https://github.com/axe2023/cryptoemperor-ai.git
cd cryptoemperor-ai
```

### 2. 安装依赖
```bash
pip3 install -r requirements.txt
```

### 3. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 Binance API Key
```

## ⚙️ 配置

编辑 `config.yaml` 调整参数：

```yaml
# 交易对列表
symbols:
  - BTCUSDT
  - ETHUSDT
  - BNBUSDT
  - SOLUSDT

# RSI 参数
rsi:
  period: 14          # RSI 计算周期
  oversold: 30        # 超卖阈值（买入信号）
  overbought: 70      # 超买阈值（卖出信号）

# K线参数
kline:
  interval: 15m       # K线周期 (1m, 5m, 15m, 1h, 4h, 1d)
  limit: 100          # 获取K线数量

# 日志配置
logging:
  level: INFO         # 日志级别
  dir: logs           # 日志目录
  max_files: 30       # 保留最近N天的日志
```

## 🎯 使用方法

### 方式 1：自动化脚本（推荐）
```bash
./run_daily.sh
```

输出：
- 实时信号到终端
- 信号记录到 `data/signals.json`
- 日报生成到 `reports/report_YYYY-MM-DD.md`
- 日志记录到 `logs/YYYY-MM-DD.log`

### 方式 2：直接运行 Python
```bash
python3 main.py
```

### 方式 3：定时调度器
```bash
python3 scheduler.py
```

按配置的时间自动运行扫描和日报生成（默认：每天 09:00/12:00/15:00/18:00/21:00 扫描，22:00 生成日报）。

### 方式 4：Docker 部署（推荐生产环境）
```bash
docker-compose up -d
```

详见 [DOCKER.md](DOCKER.md)。

### 方式 5：回测历史策略
```bash
python3 -c "
from backtest import Backtester
from binance.client import Client
import os

client = Client(os.getenv('BINANCE_API_KEY'), os.getenv('BINANCE_SECRET_KEY'))
backtester = Backtester(initial_capital=10000)

result = backtester.run_backtest(
    client=client,
    symbol='BTCUSDT',
    start_date='2024-01-01',
    end_date='2024-12-31',
    rsi_period=14,
    oversold=30,
    overbought=70
)

print(f'总交易次数: {result[\"total_trades\"]}')
print(f'胜率: {result[\"win_rate\"]:.2f}%')
print(f'总收益: {result[\"total_return\"]:.2f}%')
print(f'最大回撤: {result[\"max_drawdown\"]:.2f}%')
"
```

## 🔒 安全建议

1. **使用只读 API Key**：不要开启提现/交易权限
2. **不要提交 .env**：已添加到 `.gitignore`
3. **定期轮换密钥**：建议每月更换一次 API Key
4. **监控异常访问**：定期检查 Binance API 日志

## 📈 风控参数

编辑 `config.yaml` 添加风控配置：

```yaml
risk_control:
  max_position_pct: 10.0          # 单币种最大仓位比例
  max_total_position_pct: 50.0    # 总仓位比例
  stop_loss_pct: 5.0              # 止损比例
  daily_loss_limit_pct: 10.0      # 日亏损上限
  max_trades_per_day: 20          # 每日最大交易次数
```
